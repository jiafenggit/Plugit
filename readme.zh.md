## 关于Plugit
`Plugit`是一个基于`Koa`的服务器框架。

Plugit一词来源于`Plug`以及`git`，意在将开发流程插件化、组件化、业务可回滚化。

---
## 相关依赖
```
"co": "^4.6.0",
"koa": "^1.2.1",
"koa-body": "^1.4.0",
"koa-jwt": "^1.2.0",
"koa-router": "^5.4.0",
"koa-static": "^2.0.0",
"mongodb": "^2.2.4",
"mongoose": "^4.5.6",
"mongoose-timestamp": "^0.6.0",
"mongoose-unique-validator": "^1.0.2"
```
Plugit只是基于Koa和Mongoose的一种实现，并不是一个全新的服务器技术。
